# Imkerei-Apps & Digitale Tools

## Empfehlung: Diese 4 Apps installieren

### 1. BeeSmart (Pflicht)
- **Plattform:** iOS, Android
- **Preis:** Gratis (Basis), CHF 5.-/Monat (Premium)
- **Warum:** Einzige App spezifisch für die Schweiz. Behandlungsjournal nach CH-Gesetz, dreisprachig (DE/FR/IT), KI-basiertes BeePhone für Schwarmvorhersage.
- **Download:** [App Store](https://apps.apple.com/app/beesmart) / [Play Store](https://play.google.com/store/apps/details?id=com.beesmart)
- **Website:** [beesmart.ch](https://beesmart.ch)

### 2. BeeTraffic (Pflicht bei Wanderung)
- **Plattform:** iOS, Android, Web
- **Preis:** Gratis
- **Warum:** Offizielle App von BienenSchweiz/Identitas für gesetzlich vorgeschriebene Wandermeldung. Muss genutzt werden wenn Völker transportiert werden.
- **Website:** [beetraffic.ch](https://www.beetraffic.ch)

### 3. Varroa-App
- **Plattform:** iOS, Android
- **Preis:** Gratis
- **Warum:** Wissenschaftlich fundiertes Varroa-Management. Berechnet optimale Behandlungszeitpunkte, Nachbar-Warnsystem bei hohem Befall.
- **Download:** [App Store](https://apps.apple.com/app/varroa-app) / [Play Store](https://play.google.com/store/apps/details?id=de.varroaapp)

### 4. Waagvölker BienenSchweiz (Web)
- **Plattform:** Web (Browser-Lesezeichen)
- **Preis:** Gratis
- **Warum:** Interaktive Karte mit Echtzeit-Gewichtsdaten von Referenzvölkern in der ganzen Schweiz. Zeigt Trachtbeginn/-ende in deiner Region.
- **Website:** [bienen.ch/waagvoelker](https://www.bienen.ch/waagvoelker)

## Schweiz-spezifische Hinweise

### Gesetzliche Pflichten
- **Behandlungsjournal:** Seit 2023 gesetzlich vorgeschrieben → BeeSmart oder ApiManager nutzen
- **Wandermeldung:** Bei Transport von Völkern → BeeTraffic App (Pflicht)
- **Tierseuchenmeldung:** Faulbrut/Sauerbrut sofort melden (kantonaler Bieneninspektor)

### BienenSchweiz Mitgliedschaft
- Inkl. Versicherung (Elementarschäden, Vergiftung)
- Zugang zu Waagvölker-Daten
- Schweizerische Bienen-Zeitung
- Vergünstigungen bei Kursen
- **Empfehlung:** Unbedingt Mitglied werden!

## Fazit

Für den Start in Arosa braucht es:
1. **BeeSmart** → Völkerverwaltung & Behandlungsjournal (CH-konform)
2. **BeeTraffic** → Falls Wanderung (Transport zum Trachtgebiet)
3. **Varroa-App** → Optimales Behandlungsmanagement
4. **bienen.ch Waagvölker** → Trachtbeobachtung für GR
5. **HiveWatch App** → Kommt automatisch mit der Stockwaage
